Special Events

At most one special event may occur per expedition. They are determined by the map pattern, and each appear with the corresponding Nightlords only:

  • Morgott Invasion: Adel, Gnoster, Heolstor
  • Maris Bubbles: Gnoster, Caligo, Heolstor
  • Gnoster Plague: Maris, Libra, Heolstor
  • Libra Curse: Fulghor, Caligo, Heolstor
  • Frenzy Tower: Gnoster, Libra
  • Extra Night Boss: Adel, Fulghor
  • Night Horde: Gladius, Maris, Fulghor
  • Meteor Strike: Gladius, Adel, Caligo
  • Walking Mausoleum: Maris, Caligo
  • Difficult Sorcerer's Rise: Gnoster, Libra

Frenzy Towers, Walking Mausoleums, and difficult Sorcerer's Rises are in the map from the start. All other events are configured to occur on either Day 1 or Day 2. The game waits a random amount of time from the start of the day to trigger the event. Event-specific conditions like player positions must also be satisfied.

  • Day 1 events happen randomly 6-8 minutes in.
  • When Shifting Earth is active, Day 2 Night Boss events (italicized above) happen randomly 2-3 minutes in.
  • Otherwise, Day 2 events happen randomly 2-8 minutes in.

FAQ

Why are there so few map patterns?

Not sure, but it makes some sense to give newcomers a foothold to learn the map and its bosses if they get stuck attempting any particular Nightlord. It may have been necessary for QA. Still, the lack of pattern variety becomes evident the more you play, especially with speedruns and other challenge runs. Ideally the game should have had >5k patterns, even with limited night bosses per Nightlord. Nightreign's map pattern system has strong fundamentals which future patches hopefully build on.

What is randomized?

A 32-bit random value we can call the expedition seed determines the map pattern (including shifting earth) and the active remembrance when multiple options are available. It is generated only after matchmaking and selecting a target Nightlord.

While second circle locations are set by the pattern, the first circle location is random based on the expedition seed. The first circle's perimeter is placed at a random angle and eligible distance away from the second circle, such that it contains the second circle. The expedition seed also determines the spawn point approach direction, scarab locations, merchant shops, and the presence of cataclysms (camps with Night Invaders).

Random item and buff drops are not seeded in advance. Likewise, various in-game interactions such as special events may occur at per-run random times or locations.

Can I play other map patterns with mods?

Yes. However, unlike the certified and released game, bugs are more likely. I've created Nightreign Randomizer and Derandomizer which can produce new map patterns with random combinations of camps and bosses, with additional randomization planned. Make sure to use an alternate save file for all mods.

Can I force a certain map pattern?

Yes, this is the derandomizer part of the randomizer mod. You must use an alternate save file for this as well.
